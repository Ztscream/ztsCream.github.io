Hugo Blog Setup Guide

Introduction

Hugo is a static site generator written in Go language, featuring fast speed, powerful functionality, and ease of use.

Installing Hugo

Windows Installation

  1. Download Hugo Extended version
  2. Extract to specified directory
  3. Add to system PATH

Create New Site 

1
2

hugo new site myblog
cd myblog

Choose Theme

Hugo has rich themes to choose from, recommend using Stack theme:

1

git clone https://github.com/CaiJimmy/hugo-theme-stack.git themes/hugo-theme-stack

Configure Site

Configure site information in hugo.yaml:

1
2
3

baseurl: https://yourdomain.com
title: My Blog
theme: hugo-theme-stack

Create Content 

1

hugo new post/my-first-post.md

Local Preview 

1

hugo server -D

Deployment

GitHub Pages

  1. Create GitHub repository
  2. Configure GitHub Actions
  3. Push code for automatic deployment

Summary

Hugo is an excellent static site generator, suitable for building personal blogs, technical documentation, and other websites.
